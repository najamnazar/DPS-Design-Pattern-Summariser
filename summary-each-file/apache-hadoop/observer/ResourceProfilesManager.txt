ResourceProfilesManager acts as an observer for publisher RMActiveServiceContext, which is inherited by ResourceProfilesManagerImpl, which inits Configuration parameter of config, which gets String parameter of profile Profile, which gets Resource Profiles, which reloads Profiles, which gets Default Profile, which gets Minimum Profile, which gets Maximum Profile.
