FSDatasetMBean acts as an observer, which gets String parameter of bpid Block Pool Used, which gets Dfs Used, which gets Capacity, which gets Remaining, which gets Storage Info, which gets Num Failed Volumes, which gets Failed Storage Locations, which gets Last Volume Failure Date, which gets Estimated Capacity Lost Total, which gets Cache Used, which gets Cache Capacity, which gets Num Blocks Cached, which gets Num Blocks Failed To Cache, which gets Num Blocks Failed To Uncache, which gets Last Dir Scanner Finish Time, which gets Pending Async Deletions.
