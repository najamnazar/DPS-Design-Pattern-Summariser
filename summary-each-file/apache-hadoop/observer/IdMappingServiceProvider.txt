IdMappingServiceProvider acts as an observer for publisher WriteManager, which is inherited by ShellBasedIdMapping, which gets String parameter of user Uid, which gets String parameter of group Gid, which gets int parameter of uid, String parameter of unknown User Name, which gets int parameter of gid, String parameter of unknown Group Name, which gets String parameter of user Uid Allowing Unknown, which gets String parameter of group Gid Allowing Unknown.
