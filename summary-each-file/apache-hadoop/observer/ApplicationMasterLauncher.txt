ApplicationMasterLauncher acts as a publisher for observer Configuration, which services Configuration parameter of conf Init, which services Start, which creates RMAppAttempt parameter of application, AMLauncherEventType parameter of event Runnable Launcher, which launches RMAppAttempt parameter of application, which services Stop, which runs, which cleanups RMAppAttempt parameter of application, which handles AMLauncherEvent parameter of appEvent.
