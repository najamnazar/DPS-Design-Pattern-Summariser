AbfsOperationMetrics acts as an observer for publisher AbfsClientThrottlingAnalyzer, which is inherited by, which gets Bytes Failed, which gets Bytes Successful, which gets Operations Failed, which gets Operations Successful, which gets End Time, which sets final parameter of long End Time, which gets Start Time, which adds long parameter of bytes Bytes Failed, which adds long parameter of bytes Bytes Successful, which increments Operations Failed, which increments Operations Successful.
