AbstractFpgaVendorPlugin acts as an observer for publisher FpgaDiscoverer, which is inherited by IntelFpgaOpenclPlugin, which inits Configuration parameter of conf Plugin, which diagnoses int parameter of timeout, which discovers int parameter of timeout, which gets Fpga Type, which retrieves String parameter of id, String parameter of dstDir, Map<Path, parameter of List<String>> IPfile Path, which configures String parameter of ipPath, FpgaDevice parameter of device IP.
