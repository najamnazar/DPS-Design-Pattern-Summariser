ErasureDecoder acts as a publisher for observer Configuration, which calculates ECBlockGroup parameter of blockGroup Coding, which gets Num Data Units, which gets Num Parity Units, which gets Options, which gets ECBlockGroup parameter of blockGroup Input Blocks, which gets ECBlockGroup parameter of blockGroup Output Blocks, which prefers Direct Buffer, which releases, which prepares ECBlockGroup parameter of blockGroup Decoding Step, which gets ECBlockGroup parameter of blockGroup Num Erased Blocks, which gets ECBlock[] parameter of inputBlocks Num Erased Blocks, which gets ECBlock[] parameter of inputBlocks Erased Indexes.
