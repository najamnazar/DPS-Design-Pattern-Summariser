ContainerInfo acts as an observer for publisher ContainersInfo, which is inherited by, which gets Container Id, which gets Allocated MB, which gets Allocated VCores, which gets Assigned Node Id, which gets Priority, which gets Started Time, which gets Finished Time, which gets Elapsed Time, which gets Diagnostics Info, which gets Log Url, which gets Container Exit Status, which gets Container State, which gets Node Http Address, which gets Node Id, which gets Allocated Resources, which gets Exposed Ports, which has Custom Resources.
