MicroZookeeperService acts as a publisher for observer Configuration, which gets Connection String, which gets Connection Address, which gets int parameter of port Address, which services Configuration parameter of conf Init, which mkdirs File parameter of dir Strict, which adds String parameter of text, Object... parameter of args Diagnostics, which gets Diagnostics, which setups Security, which services Start, which services Stop, which supplies Binding Information, which gets Random Available Port.
