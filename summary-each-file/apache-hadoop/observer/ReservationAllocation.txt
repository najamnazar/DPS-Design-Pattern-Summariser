ReservationAllocation acts as an observer for publisher InMemoryPlan, which is inherited by InMemoryReservationAllocation, which gets Reservation Id, which gets Reservation Definition, which gets Start Time, which gets End Time, which gets Allocation Requests, which gets Plan Name, which gets User, which contains Gangs, which sets long parameter of acceptedAt Acceptance Timestamp, which gets Acceptance Time, which gets long parameter of tick Resources At Time, which gets Resources Over Time, which gets long parameter of start, long parameter of end Resources Over Time, which gets Periodicity, which sets long parameter of period Periodicity.
