ITestNativeAzureFileSystemClientLogging acts as a publisher for observer AzureBlobStorageTestAccount, which verifies String parameter of capturedLogs, String parameter of entity Storage Client Logs, which updates Boolean parameter of loggingFlag File System Configuration, which performs WASBOperations, which tests Logging Enabled, which gets LogCapturer parameter of logs Log Output, which tests Logging Disabled, which creates Test Account.
