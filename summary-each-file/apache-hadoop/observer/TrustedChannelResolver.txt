TrustedChannelResolver acts as a publisher for observer Configuration, which gets Configuration parameter of conf Instance, which sets Configuration parameter of conf Conf, which gets Conf, which is Trusted, which is InetAddress parameter of peerAddress Trusted.
TrustedChannelResolver acts as a publisher for observer Configuration, which is inherited by WhitelistBasedTrustedChannelResolver, BlackListBasedTrustedChannelResolver, which gets Configuration parameter of conf Instance, which sets Configuration parameter of conf Conf, which gets Conf, which is Trusted, which is InetAddress parameter of peerAddress Trusted.
