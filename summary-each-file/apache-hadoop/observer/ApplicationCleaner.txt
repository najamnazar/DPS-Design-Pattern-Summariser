ApplicationCleaner acts as a publisher for observer Configuration, which is inherited by DefaultApplicationCleaner, which inits Configuration parameter of config, GPGContext parameter of context, which gets GPGContext, which gets Registry Client, which gets Apps From Router, which gets Router Known Applications, which runs.
