CsiAdaptorProtocol acts as an observer for publisher VolumeManagerImpl, which is inherited by CsiAdaptorPlugin, CsiAdaptorProtocolService, CsiAdaptorProtocolPBClientImpl, which gets GetPluginInfoRequest parameter of request Plugin Info, which validates ValidateVolumeCapabilitiesRequest parameter of request Volume Capacity, which nodes NodePublishVolumeRequest parameter of request Publish Volume, which nodes NodeUnpublishVolumeRequest parameter of request Unpublish Volume.
