InstantiationAwareBeanPostProcessor acts as an observer, which posts Class<?> parameter of beanClass, String parameter of beanName Process Before Instantiation, which posts Object parameter of bean, String parameter of beanName Process After Instantiation, which posts PropertyValues parameter of pvs, Object parameter of bean, String parameter of beanName Process Properties, which posts PropertyValues parameter of pvs, PropertyDescriptor[] parameter of pds, Object parameter of bean, String parameter of beanName Process Property Values.
