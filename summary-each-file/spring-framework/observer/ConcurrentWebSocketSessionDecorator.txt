ConcurrentWebSocketSessionDecorator acts as a publisher for observer WebSocketSession, which gets Send Time Limit, which gets Buffer Size Limit, which gets Buffer Size, which gets Time Since Send Started, which sets Consumer<WebSocketMessage<?>> parameter of callback Message Callback, which sends WebSocketMessage<?> parameter of message Message, which shoulds Not Send, which tries Flush Message Buffer, which checks Session Limits, which limits String parameter of reason Exceeded, which closes CloseStatus parameter of status, which tos String.
