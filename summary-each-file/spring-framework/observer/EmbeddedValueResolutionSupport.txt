EmbeddedValueResolutionSupport acts as a publisher for observer StringValueResolver, which sets StringValueResolver parameter of resolver Embedded Value Resolver, which resolves String parameter of value Embedded Value.
EmbeddedValueResolutionSupport acts as a publisher for observer StringValueResolver, which is inherited by Jsr310DateTimeFormatAnnotationFormatterFactory, NumberFormatAnnotationFormatterFactory, Jsr354NumberFormatAnnotationFormatterFactory, DateTimeFormatAnnotationFormatterFactory, which sets StringValueResolver parameter of resolver Embedded Value Resolver, which resolves String parameter of value Embedded Value.
