TransactionDefinition acts as an observer for publisher CallCountingTransactionManager, NoSynchTransactionManager, SynchTransactionManager, which is inherited by DelegatingTransactionDefinition, DefaultTransactionDefinition, TransactionAttribute, StaticTransactionDefinition, ResourceTransactionDefinition, which gets Propagation Behavior, which gets Isolation Level, which gets Timeout, which is Read Only, which gets Name, which withes Defaults.
