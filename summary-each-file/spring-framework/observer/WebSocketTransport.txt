WebSocketTransport acts as an observer, which gets Web Socket Client, which gets Transport Types, which connects TransportRequest parameter of request, WebSocketHandler parameter of handler, which starts, which stops, which is Running, which tos String, which afters WebSocketSession parameter of webSocketSession Connection Established, which handles WebSocketSession parameter of webSocketSession, TextMessage parameter of message Text Message, which handles WebSocketSession parameter of webSocketSession, Throwable parameter of ex Transport Error, which afters WebSocketSession parameter of webSocketSession, CloseStatus parameter of status Connection Closed.
