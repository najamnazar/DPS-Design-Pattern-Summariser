UndertowSecurityProvider acts as an observer for publisher UndertowComponent, which is inherited by ElytronSecurityProvider, SpringSecurityProvider, which adds BiConsumer<String, parameter of Object>, HttpServerExchange parameter of httpExchange Header, which authenticates HttpServerExchange parameter of httpExchange, List<String> parameter of allowedRoles, which accepts Object parameter of configuration, String parameter of endpointUri Configuration, which wraps HttpHandler parameter of httpHandler Http Handler, which requires Servlet Context.
