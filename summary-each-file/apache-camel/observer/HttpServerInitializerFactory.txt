HttpServerInitializerFactory acts as a publisher for observer Channel, which is inherited by HttpServerSharedInitializerFactory, which creates NettyConsumer parameter of nettyConsumer Pipeline Factory, which inits Channel parameter of ch Channel, which creates CamelContext parameter of camelContext, NettyServerBootstrapConfiguration parameter of configuration SSLContext, which configures Server SSLOn Demand, which supports Compressed.
