ManagedTracer acts as a caretaker for memento Context, which inits ManagementStrategy parameter of strategy, which gets Context, which gets Tracer, which gets Enabled, which gets Camel Id, which gets Camel Management Name, which is Standby, which sets boolean parameter of enabled Enabled, which is Enabled, which sets String parameter of pattern Trace Pattern, which gets Trace Pattern, which gets Trace Counter, which resets Trace Counter.
