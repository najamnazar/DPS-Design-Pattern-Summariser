CxfSpringRouterTest acts as a caretaker for memento Context, which gets Router Address, which gets Service Address, which configures ServerFactoryBean parameter of svrBean Factory, which starts Service, which shutdowns Service, which sets Up, which tears Down, which creates Route Builder, which creates Application Context, which gets CXFClient, which tests Invoking Service From CXFClient, which tests Onway Invocation.
